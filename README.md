BMI Calculator Microservice

This is a simple BMI Calculator built using Python and Flask.

It calculates BMI using weight and height given in the URL and returns the result in JSON format.

Live URL

https://bmi-flask-microservice.onrender.com/

GitHub Repository

https://github.com/KowsiAss/bmi-flask-microservice

How to Use

Open in browser:

/bmi?weight=70&height=1.75

Example:

https://bmi-flask-microservice.onrender.com/bmi?weight=70&height=1.75
How to Run Locally

Install Flask:

pip install flask

Run:

python app.py

Open:

http://127.0.0.1:5000/
